## Getting started

#### Requirements

- Xcode
- `yarn` (or `npm`)

#### Installation

```sh
git clone https://github.com/jclerc/moodie-ios.git
cd moodie-ios
yarn # or `npm i`
yarn start # or `npm start`
open -a Xcode ios/Moodie.xcodeproj
```

Then build the project right from Xcode and run it.
